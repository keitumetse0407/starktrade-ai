# Deploy Backend to Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/zZBqYJ?referralCode=starktrade)

Or deploy manually:

1. Go to https://railway.app/new
2. Click "Deploy from GitHub repo"
3. Select `keitumetse0407/starktrade-ai`
4. Set root directory to `/backend`
5. Add PostgreSQL
6. Add Redis
7. Done
